# LLM Lab Prep

This repo is for learning the basics needed to understand and build small LLM-related experiments.

The first focus area is Python and PyTorch:

- tensors
- gradients
- loss functions
- simple linear regression

## Getting Started

Install PyTorch, then run the first autograd example:

```bash
python 00-python-pytorch/gradient_test.py
```

The goal is to build intuition step by step. Start with small examples, run them, change one thing, and run them again.
